
To use the texture synthesis algorithm some external libraries should be compiled and installed using the 
creaTools

crea
creaANN
creaMaracasVisu

Use cmake to configure the variables where you installed this libraries and configure the project.
When you are done configuring the project type make

The project will generate an executable called optimizationTexture
type ./optimizationTexture for an example on how to use the program.
